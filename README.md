# Tronlix Web V2

Architecture-First Web Engineering.

## Stack

- React 19 + TypeScript (strict)
- Vite 6
- Tailwind CSS v4
- GSAP 3 + ScrollTrigger
- React Router v7

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run preview
```
